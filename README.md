**Sales Performance Dashboard**

**Project Overview**
Business Intelligence dashboard developed using Microsoft Power BI to analyze sales performance, profitability, customer behavior, product insights, and geographic trends.

**Business Objectives**
The dashboard provides insights into:
-	Overall sales and profitability performance
-	Monthly sales trends and growth patterns
-	Product category and product-level performance
-	Customer segmentation and purchasing behavior
-	Regional sales distribution and profitability
-	Return rate analysis

**Tools & Technologies**
-	Microsoft Power BI
-	Power Query
-	DAX
-	Data Modeling
-	CSV Data Sources

**Dashboard Pages**
1.	Executive Overview
Provides business performance visibility with:
-	Total Sales
-	Total Profit
-	Profit Margin
-	Orders and Customers
-	Monthly Sales Trends
-	Category Performance
-	Country Performance
-	Return Analysis

2.	Product Analytics analyzes:
-	Product sales distribution
-	Top performing products
-	Product profitability
-	Quantity sold analysis
-	Category return rates

3.	Customer Analytics focuses on:
-	Customer segmentation
-	Customer purchasing patterns
-	Sales by occupation and income group
-	Top customers
-	Return behavior by customer demographics

4.	Territory Analytics provides geographic insights including:
-	Global sales distribution
-	Sales by country
-	Profit contribution by country
-	Customer distribution
-	Return rates across regions

**Data Model**
Uses a star schema approach consisting of:
Fact Tables
-	Sales Transactions
-	Returns

**Dimension Tables**
-	Customer
-	Product
-	Product Category
-	Product Subcategory
-	Territory
-	Calendar

**Major DAX measures Implemented**
-	Total Sales
-	Total Profit
-	Return Rate %
-	Average Selling Price
-	Customer Metrics
-	Time Intelligence Calculations
-	Dynamic Dashboard Titles
